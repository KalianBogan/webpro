Особые возможности

  • 1. Мультиязычность. Сайт поддерживает несколько языков, что позволяет сделать информацию доступной для большего количества людей.
  • 2. Режим отображения для слабовидящих позволяет отображать только основную информацию, а также изменить размер шрифта или отображение его на черном фоне.
  • 3. Страница сайта в котором отражены ссылки на все важные страницы сайта.
  • 4. Позволяет адаптировать и подготовить интересующую вас страницу для печати.
  • 5. Быстрый поиск интересующей информации на сайте
