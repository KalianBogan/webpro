Описание
Агрегаттор туристических путевок обладает конструктором сайтов, где пользователь системы способен создать свой сайт для турфирмы, где уже будут все туры сайта агрегатора. Также есть возможность полностью редактировать свой сайт.

Два вида корзины с нестандартной системой скидок в зависимости от выбранного вами тура.
Корзина для пакетных туров позволяет выбрать фиксированую дату отправления из доступных, а также дату возвращения
Корзина для тура на своем автомобиле позволяет выбрать любую дату отправления, которая вам подходит
